Python开发轻量级爬虫

发表于 2017-06-06 |

使用Python(3)开发一个轻量级爬虫，实现不需要登录的静态网页的抓取。

爬虫是什么？有什么价值？

爬虫是一段自动抓取互联网信息的程序。将互联网数据，为我所用。

简单爬虫架构

爬虫模块 功能
URL管理器 对将要爬取的URL和已经爬取过的URL这两个数据进行管理
网页下载器 将URL对应的网页下载，存储为一个字符串，并传送给网页解析器进行解析
网页解析器 解析出有价值的数据，和指向其他页面的URL，并将其补充进URL管理器

以上三个模块构成了一个爬虫程序，在其基础上编写一个爬虫调度端，用来启动、执行、停止爬虫或者监视爬虫的运行情况，以及一个输出数据模块。这些模块就组成了一个简单的爬虫架构，该架构就能将互联网中所有的网页抓取下来。

爬虫调度端

首先在一个包里面创建spider_main.py、url_manager.py、html_downloader.py、html_parser.py、html_outputer.py5个python文件，其中spider_main.py就是爬虫的调度程序，前面已经知道爬虫调度端是用来启动、执行、停止爬虫或者监视爬虫的运行情况，所以在文件中需要给出一个入口URL，一个满足条件就一直执行的while循环语句，一个停止循环的条件和打印当前爬虫运行情况的语句。下面来看看该怎么编写。

URL管理器url_manager.py

用来管理待抓取URL集合和已抓取URL集合，可以防止重复抓取、循环抓取。本次使用python内存来实现功能，使用set来储存数据（set可以取出重复的数据）。除了内存外，还可以使用关系数据库和缓存数据库。

下载器html_downloader.py

将互联网上URL对应的网页下载到本地的工具。本次爬取的是不需要登录的静态网页，所以可以使用最简单的urllib.request.urlopen的方法实现网页的下载。而有的需要添加data、http header，甚至需要添加特殊情景的处理器时，可以参考这里的方法实现。

解析器html_parser.py

从网页中提取有价值数据和URL的工具。本次使用Beautiful Soup作为网页解析器，因为其功能强大，可以使用python自带的html.parser作为解析工具，也可以使用lxml作为解析工具，同时可以使用正则表达式作为补充。Beautiful Soup的用法

输出器html_outputer.py

运行程序

编写完以上5个模块后，运行调度程序spider_main.py，等待几分钟就得到一个output.html的文件。

复制该文件的路径，在浏览器中打开，就能看到如下图所示的结果了。有时候在浏览器中显示的是乱码，那么就在查看选项下改文字编码为Unicode，就可以看到正常的结果了。
